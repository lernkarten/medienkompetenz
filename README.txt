Medienkompetenz-Karten ab Klasse 4 (9+)

Dieses Projekt stellt ein frei nutzbares Kartenset zur Medienkompetenz für Kinder (ab Klasse 4) bereit.
Die Karten behandeln zentrale Themen wie Datenschutz, Informationen im Internet, KI, Rechte sowie reflektierte Mediennutzung im Alltag.

Dieses Repository enthält die statische Website, erreichbar unter: https://medienkompetenz-karten.de/,
und die Kartensets in PDF-Form.
Die Website wird über GitHub Pages bereitgestellt.

Inhalt
- Startseite: index.html
- Kartenansicht: karten.html
- Projektseite: ueber-das-projekt.html
- Rechtliches: impressum.html, datenschutz.html

Downloads
- Die Kartensets in PDF-Form liegen im Ordner: pdf/.
- Verfügbar sind Varianten für Druck zu Hause oder Schul-Drucker sowie für professionellen Druck (mit Beschnitt).

Lokale Vorschau
- Die Website kann direkt über index.html im Browser geöffnet werden.

Lizenz
- Die verbindlichen Nutzungsbedingungen stehen in der Datei LICENSE.
- Kurzfassung auf der Website: Bereich "Lizenz" auf der Startseite.

Hinweis
- Diese Inhalte sind für private, bildungsbezogene und nicht-kommerzielle Nutzung gedacht.
